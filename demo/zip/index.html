Zip ファイルをクライアントサイドで作成

JavaScript を使った Zip ファイルの構築を行います。
Web Workers, Typed Array, Blob (もしくは Data URL), download 属性などに対応しているブラウザで正常に動作します。 Safari や Firefox などではダウンロード完了後にファイル名を変更してください。

このページで使用しているテキストファイルは青空文庫のものを使用しています。
